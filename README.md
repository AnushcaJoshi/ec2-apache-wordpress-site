# cloud-web-hosting-project

## Project Overview:
### - Goal: 
Create a virtual environment to host a website.
### - Technologies Used: 
#### AWS EC2
#### SSH (MobaXterm)
#### Linux OS
#### Apache server
#### WordPress.
