## Project Overview:
### - Goal: 
Hosting a WordPress Web Application on AWS EC2 Service using Apache Server.

### - Technologies Used: 
#### AWS EC2 (Ubuntu)
#### Elastic IP (AWS) 
#### SSH Client: MobaXterm 
#### Apache server
#### MySQL Server
#### WordPress
